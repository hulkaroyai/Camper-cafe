#Camper Cafe
**Camper Cafe** is a simple and elegant cafe menu webpage created as part of the FreeCodeCamp Curriculum.
##Description 
This webpage displays a ficttional cafe menu with items and prices.It's designed to be ahands-on exercise for applying fundational web development concepts.The layout is straightforward but provides browsing experience across different devices.
##Features
-**Menu Section:**Lists coffee and dessert items
-**Pricing Table:**Clearly displays the price for each item.
-**Responsive Design:**Adapts seamlessy to mobile, tablet, and desktop screens.
-**Clean Interface:**Easy to read and navigate.
